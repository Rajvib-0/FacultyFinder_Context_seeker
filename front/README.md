# FacultyFinder - AI-Powered Faculty Search

Find faculty members based on research interests using semantic search.

## Features
-  AI-powered semantic search
-  Hybrid search (semantic + keyword)
-  Fast results with FAISS indexing
- Rich faculty profiles

## Usage
1. Go to the "Search" page
2. Enter your research interests
3. Get matched with relevant faculty

## Technology
- Streamlit for UI
- Sentence Transformers for embeddings
- FAISS for vector search
